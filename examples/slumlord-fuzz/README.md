# slumlord-fuzz — Phase 4 protocol-variety harness

[Slumlord](https://github.com/igneous-labs/slumlord) — a simple
zero-fee SOL flash loan program by Igneous Labs (Sanctum team).
Selected Day 40 as the Phase 4 N=1 target for the protocol-variety
hypothesis test.

See [`docs/slumlord-ix-inventory.md`](../../docs/slumlord-ix-inventory.md)
for the per-ix surface analysis. Phase 4 protocol-variety stopping
rule: 2 protocol-size negatives close the axis and bind the
protocol-variety pivot.

## Pre-fuzz honest framing

Slumlord is a **competently-coded simple program**, not hackathon-grade
code. The "less-hardened" axis being tested here is *protocol size +
TVL*, not *code quality*. Each ix has explicit
`*_verify_account_keys` + `*_verify_account_privileges` helpers
generated by solores from the Shank IDL — functionally equivalent
to Anchor's `Account<'info, T>` constraints for account-validation
invariants.

Best honest pre-fuzz estimate: **0 violations across all 5
applicable invariants**.

## External program dependency

Slumlord source not vendored. Clone separately + build:

```bash
git clone https://github.com/igneous-labs/slumlord.git ~/src/slumlord

cd ~/src/slumlord/slumlord
SDKROOT=$(xcrun --show-sdk-path) \
CFLAGS="-isysroot $(xcrun --show-sdk-path)" \
cargo build-sbf --tools-version v1.39
```

Produces `~/src/slumlord/target/deploy/slumlord.so` (~115KB).
The harness loads via absolute path — see `SLUMLORD_SO_PATH` in
`fuzz/slumlord/src/main.rs`. Update if cloned elsewhere.

Same `SDKROOT + platform-tools v1.39` incantation as klend Day 17 —
required on macOS to compile the old `ahash` + `solana-frozen-abi`
deps that Slumlord's toolchain pin (rust 1.68.0 / solana 1.16.20)
needs.

## Wire format

Native (non-Anchor). 1-byte u8 discriminator at `data[0]` per
`idl.json`:

| Disc | ix | args |
|---|---|---|
| 0 | Init | none |
| 1 | Borrow | none |
| 2 | Repay | none |
| 3 | CheckRepaid | none |

No ix takes args. All state is account-driven (slumlord PDA
balance + `old_lamports` field).
